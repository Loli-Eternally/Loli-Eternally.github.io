三个量子数的由来

前言

前面介绍了球谐函数，里面包括了两个量子数 lll 和 mmm，它们分别代表角动量量子数和磁量子数，因为拉普拉斯算子在球坐标下可写为：

∇2=1r2∂∂r ⁣(r2∂∂r)+1r2∇Ω2,(1)\nabla^2 = \frac{1}{r^2}\frac{\partial}{\partial r}\!\left(r^2 \frac{\partial}{\partial r}\right) + \frac{1}{r^2}\nabla_\Omega^2,\tag{1} ∇2=r21​∂r∂​(r2∂r∂​)+r21​∇Ω2​,(1)

其中 ∇Ω2\nabla_\Omega^2∇Ω2​ 仅作用在角变量 (θ,ϕ)(\theta,\phi)(θ,ϕ)。

梯度算子 ∇\nabla∇ 本质是一个动量算符，因为 ∇\nabla∇ 是作用在空间维度上的微分算子，它的作用是将空谐函数 eik⋅xe^{ik\cdot x}eik⋅x 中的空间系数取下来，因此本征值是 ikikik，这也是为什么动量算符 p=ℏk=−iℏ⋅∇p=\hbar k=-i\hbar\cdot \nablap=ℏk=−iℏ⋅∇ 的原因。拉普拉斯算子 Δ\DeltaΔ 亦或写成 ∇2\nabla^2∇2，本质是一个能量算符，因为动能的动量表示式为 Ek=p2/2m=−ℏ22m∇2E_k=p^2/2m=-\frac{\hbar^2}{2m}\nabla^2Ek​=p2/2m=−2mℏ2​∇2。至于角动量算符，我们知道动量 p=mvp=mvp=mv，角动量 L=r×p=IωL=r\times p= I\omegaL=r×p=Iω，其中 I=∫dm⋅r2I=\int dm\cdot r^2I=∫dm⋅r2 为转动惯量。这里能量包含平动动能与转动动能，平动指的是沿 rrr 径向运动的动能，转动指的是沿立体角角向运动的动能，因此 ∇Ω\nabla_{\Omega}∇Ω​ 本质是一个角动量算符， ∇Ω2\nabla_{\Omega}^2∇Ω2​ 本质是一个角动能算符。

角动能决定了粒子做圆周运动的快慢或者剧烈程度，但这是有前提的！前提就是在同一个半径 rrr 下。如果 rrr 都不一样，你谈速度就毫无意义，这就引出了第三个量子数，即由径向长度决定能级的主量子数 nnn。

主量子数是在求解三维薛定谔方程或波动方程中出现的，本质也是在求解径向方程/连带拉盖尔方程中出现的级数终止阶数。

从三维薛定谔方程出发

以氢原子为例（中心势 V(r)=−e24πε0rV(r) = -\frac{e^2}{4\pi\varepsilon_0 r}V(r)=−4πε0​re2​）：

−ℏ22m∇2ψ+V(r)ψ=Eψ.(2)-\frac{\hbar^2}{2m}\nabla^2\psi + V(r)\psi = E\psi.\tag{2} −2mℏ2​∇2ψ+V(r)ψ=Eψ.(2)

把波函数分离为：

ψ(r,θ,φ)=R(r) Ylm(θ,φ).(3)\psi(r,\theta,\varphi) = R(r)\,Y_l^m(\theta,\varphi).\tag{3} ψ(r,θ,φ)=R(r)Ylm​(θ,φ).(3)

代入并利用球坐标下的拉普拉斯算子：

∇2=1r2ddr ⁣(r2ddr)−l(l+1)r2.(4)\nabla^2 = \frac{1}{r^2}\frac{d}{dr}\!\left(r^2\frac{d}{dr}\right) - \frac{l(l+1)}{r^2}.\tag{4} ∇2=r21​drd​(r2drd​)−r2l(l+1)​.(4)

得到径向方程：

−ℏ22m[1r2ddr ⁣(r2dRdr)−l(l+1)r2R]+V(r)R=ER.(5)-\frac{\hbar^2}{2m}\left[ \frac{1}{r^2}\frac{d}{dr}\!\left(r^2\frac{dR}{dr}\right) - \frac{l(l+1)}{r^2}R \right] + V(r)R = E R.\tag{5} −2mℏ2​[r21​drd​(r2drdR​)−r2l(l+1)​R]+V(r)R=ER.(5)

乘上 −2mℏ2-\frac{2m}{\hbar^2}−ℏ22m​，整理成标准形式：

d2Rdr2+2rdRdr+[2mℏ2(E−V(r))−l(l+1)r2]R=0.(6)\frac{d^2R}{dr^2} + \frac{2}{r}\frac{dR}{dr} + \left[\frac{2m}{\hbar^2}(E - V(r)) - \frac{l(l+1)}{r^2}\right]R = 0.\tag{6} dr2d2R​+r2​drdR​+[ℏ22m​(E−V(r))−r2l(l+1)​]R=0.(6)

为了去掉一阶导项，令：

R(r)=u(r)r.(7)R(r) = \frac{u(r)}{r}.\tag{7} R(r)=ru(r)​.(7)

代入得：

d2udr2+[2mℏ2(E−V(r))−l(l+1)r2]u=0.(8)\frac{d^2u}{dr^2} + \left[\frac{2m}{\hbar^2}(E - V(r)) - \frac{l(l+1)}{r^2}\right]u = 0.\tag{8} dr2d2u​+[ℏ22m​(E−V(r))−r2l(l+1)​]u=0.(8)

回顾之前求解的三维球贝塞尔方程

u′′+[k2−l(l+1)r2]u=0(9)u''+[k^2-\frac{l(l+1)}{r^2}]u=0\tag{9} u′′+[k2−r2l(l+1)​]u=0(9)

不能说很像，它就是一模一样，只需要令 k=2m(E−V(r))ℏk=\frac{\sqrt{2m(E-V(r))}}{\hbar}k=ℏ2m(E−V(r))​​，也就是说，薛定谔方程和亥姆霍兹方程有着统一的微分方程形式（同构）。

但它们俩又有着本质的区别，即薛定谔方程中的“势能项 V(r)V(r)V(r)”改变了局部波数 k(r)k(r)k(r)，对应于一个“有效介质折射率”或“有效势场”。亥姆霍兹方程的 kkk 则通常是常数，描述均匀介质中的波。

薛定谔方程是物质波的波动方程，其本征函数是粒子的定态波函数；
亥姆霍兹方程是线性波（包括电磁波）的简谐场分布满足的方程，其本征函数是场的空间模式。

将对库仑势 V(r)=−e24πε0rV(r)=-\frac{e^2}{4\pi\varepsilon_0 r}V(r)=−4πε0​re2​ 代入：

d2udr2+[−2mEℏ2+2me24πε0ℏ21r−l(l+1)r2]u=0.(10)\frac{d^2u}{dr^2} + \left[-\frac{2mE}{\hbar^2} + \frac{2m e^2}{4\pi\varepsilon_0\hbar^2}\frac{1}{r} - \frac{l(l+1)}{r^2}\right]u = 0.\tag{10} dr2d2u​+[−ℏ22mE​+4πε0​ℏ22me2​r1​−r2l(l+1)​]u=0.(10)

我们定义几个常用参数：

κ=−2mEℏ(E<0),a0=4πε0ℏ2me2.(11)\kappa = \frac{\sqrt{-2mE}}{\hbar} \quad (\text{E<0}), \quad a_0 = \frac{4\pi\varepsilon_0\hbar^2}{me^2}.\tag{11} κ=ℏ−2mE​​(E<0),a0​=me24πε0​ℏ2​.(11)

方程变成：

d2udr2+[−κ2+2a0r−l(l+1)r2]u=0.(12)\frac{d^2u}{dr^2} + \left[-\kappa^2 + \frac{2}{a_0 r} - \frac{l(l+1)}{r^2}\right]u = 0.\tag{12} dr2d2u​+[−κ2+a0​r2​−r2l(l+1)​]u=0.(12)

其中 κ\kappaκ 为代表束缚态的衰减系数，a0a_0a0​ 为归一化半径（玻尔半径）

处理边界条件（无穷远与原点）

  1. 当 r→∞r\to\inftyr→∞：势项可忽略，方程 ⇒u′′−κ2u=0\Rightarrow u'' - \kappa^2 u = 0⇒u′′−κ2u=0，⇒ u∼e−κru \sim e^{-\kappa r}u∼e−κr（保证归一化）。
  2. 当 r→0r\to 0r→0：主导项是 −l(l+1)r2u-\frac{l(l+1)}{r^2}u−r2l(l+1)​u，⇒ u∼rl+1u \sim r^{l+1}u∼rl+1（保证有限）。

因此设：

u(r)=rl+1e−κrf(r),(13)u(r) = r^{l+1} e^{-\kappa r} f(r),\tag{13} u(r)=rl+1e−κrf(r),(13)

其中 f(r)f(r)f(r) 是待定函数，用于捕捉中间区域的修正。

代入并化简后得到关于无量纲变量

ρ=2κr,(14)\rho = 2\kappa r,\tag{14} ρ=2κr,(14)

的方程 f(ρ)f(\rho)f(ρ)：

ρf′′+2(l+1−ρ/2)f′+(1κa0−l−1)f=0.(15)\rho f'' + 2(l+1-\rho/2)f' + \left(\frac{1}{\kappa a_0} - l - 1\right)f = 0.\tag{15} ρf′′+2(l+1−ρ/2)f′+(κa0​1​−l−1)f=0.(15)

这个方程与 关联拉盖尔方程 (associated Laguerre equation) 等价。

幂级数法求解（出现主量子数）

我们设 f(ρ)=∑n=0∞anρnf(\rho) = \sum_{n=0}^{\infty} a_n \rho^nf(ρ)=∑n=0∞​an​ρn，代入后得到递推关系：

an+1=n+l+1−1κa0(n+1)(n+2l+2)an.(16)a_{n+1} = \frac{n+l+1 - \frac{1}{\kappa a_0}}{(n+1)(n+2l+2)}a_n.\tag{16} an+1​=(n+1)(n+2l+2)n+l+1−κa0​1​​an​.(16)

为了避免发散（保证波函数在无穷远处有限），级数必须终止：存在某个 nrn_rnr​ 使得分子为零。

于是：

nr+l+1=1κa0.(17)n_r + l + 1 = \frac{1}{\kappa a_0}.\tag{17} nr​+l+1=κa0​1​.(17)

我们定义：

n=nr+l+1,(18)n = n_r + l + 1,\tag{18} n=nr​+l+1,(18)

称为主量子数（principal quantum number）。

于是：

κ=1na0.(19)\kappa = \frac{1}{n a_0}.\tag{19} κ=na0​1​.(19)

因为 κ2=−2mE/ℏ2\kappa^2 = -2mE/\hbar^2κ2=−2mE/ℏ2，所以：

En=−me42(4πε0)2ℏ21n2=−EHn2,(20)E_n = -\frac{m e^4}{2(4\pi\varepsilon_0)^2\hbar^2}\frac{1}{n^2} = -\frac{E_H}{n^2},\tag{20} En​=−2(4πε0​)2ℏ2me4​n21​=−n2EH​​,(20)

其中 EHE_HEH​ 是氢原子基态能量（约 13.6 eV）。

总结

三种量子数体系的来源

量子数 来源方程 数学意义 物理意义
mmm 方位角方程 d2Φ/dφ2+m2Φ=0d^2\Phi/d\varphi^2 + m^2\Phi = 0d2Φ/dφ2+m2Φ=0 周期性约束 角动量 z 分量
lll 极角方程（连带勒让德） 有限性约束 角动量大小
nnn 径向方程（拉盖尔多项式） 级数终止 能级（主量子数）

这里就能解释高中的结构化学里，原子的电子排布式为什么要写成 Cu:1s22s22p63s23p63d104s1\text{Cu}:1s^22s^22p^63s^23p^63d^{10}4s^1Cu:1s22s22p63s23p63d104s1 这种形式。主量子数决定了主能级，在多电子原子中，同一主量子数 nnn 下，由于电子间相互作用、自旋轨道耦合等效应，不同 lll（即 s、p、d、f 轨道）会出现能级劈裂，分成 spdf 等电子轨道。不同的电子轨道有着不同的球谐函数分布图，能量自然会有所不同。电子在排布的时候会优先排在能量最低的轨道上，并遵循洪特规则和泡利不相容原理。

附录一些证明

为什么 mmm 叫做磁量子数，为什么它是角动量的 z 分量。

首先解释为什么 mmm 叫做磁量子数，是因为在经典电磁学里，带电粒子角动量的 z 分量决定磁矩在外磁场 Bz 中的能量分裂：

U=−μzB=−e2meLzB(21)U = -\mu_z B = -\frac{e}{2m_e} L_z B\tag{21} U=−μz​B=−2me​e​Lz​B(21)

量子化后：

Lz=mℏ(22)L_z = m\hbar\tag{22} Lz​=mℏ(22)

于是外磁场中能级按照 m 不同而劈裂（Zeeman 效应）：

ΔE=mℏe2meB(23)\Delta E = m\hbar\frac{e}{2m_e}B\tag{23} ΔE=mℏ2me​e​B(23)

所以 mmm 叫做“磁量子数”（magnetic quantum number）。它不是因为“有磁性”，而是因为最早是从磁场能级劈裂中发现的。

那凭什么说它就是角动量的 z 分量呢？

从形式上就可以看出，当 rrr 固定时，∇2=1r2∇Ω2\nabla^2 = \frac{1}{r^2}\nabla_\Omega^2∇2=r21​∇Ω2​，角动能算符为 EΩ=(−iℏ⋅∇)2/(2m)=ℏ2⋅1r2∇Ω2/(2m)E_\Omega=(-i\hbar\cdot \nabla)^2/(2m)=\hbar^2\cdot\frac{1}{r^2}\nabla_\Omega^2/(2m)EΩ​=(−iℏ⋅∇)2/(2m)=ℏ2⋅r21​∇Ω2​/(2m)。由于 ∇Ω2\nabla_\Omega^2∇Ω2​ 的本征值是 l(l+1)l(l+1)l(l+1)

L2=(r×p)2=(r×2mEΩ)2=ℏ2⋅∇Ω2=l(l+1)ℏ2(24)L^2=(r\times p)^2=(r\times \sqrt{2mE_\Omega})^2=\hbar^2\cdot\nabla_\Omega^2=l(l+1)\hbar^2\tag{24} L2=(r×p)2=(r×2mEΩ​​)2=ℏ2⋅∇Ω2​=l(l+1)ℏ2(24)

所以角动量的本征值是 l(l+1)ℏ\sqrt{l(l+1)}\hbarl(l+1)​ℏ，（8）可以写成

{d2udr2+1ℏ2[p^2−L^2r2]u=0p^=2m(E−V(r))L^=l(l+1)ℏ(25)\left\{ \begin{aligned} &\frac{d^2u}{dr^2} + \frac{1}{\hbar^2}\left[\hat{p}^2 - \frac{\hat{L}^2}{r^2}\right]u = 0\\ &\hat{p}=\sqrt{2m(E-V(r))}\\ &\hat{L}=\sqrt{l(l+1)}\hbar \end{aligned} \right.\tag{25} ⎩⎨⎧​​dr2d2u​+ℏ21​[p^​2−r2L^2​]u=0p^​=2m(E−V(r))​L^=l(l+1)​ℏ​(25)

由此可见，角动量是与 ℏ\hbarℏ 成正比的。

同样的方式我们对 LzL_zLz​ 进行计算

Lz=r⊥z×p⊥z=ℏ⋅r×k=ℏ⋅r∂ir∂φ=ℏi⋅∂∂φ(26)L_z=r_{\perp z}\times p_{\perp z}=\hbar\cdot r\times k=\hbar\cdot r\frac{\partial}{ir\partial\varphi}=\frac{\hbar}{i}\cdot\frac{\partial}{\partial\varphi}\tag{26} Lz​=r⊥z​×p⊥z​=ℏ⋅r×k=ℏ⋅rir∂φ∂​=iℏ​⋅∂φ∂​(26)
