亥姆霍兹方程格林函数的推导与统一

前面已经推导出了球坐标下亥姆霍兹方程完整的形式解

G(r,r′)=ik∑l=0∞∑m=−lljl(kr<) hl(1)(kr>) Ylm(θ,ϕ)Ylm∗(θ′,ϕ′),=∑l=0∞∑m=−llgl(r,r′) Ylm(θ,ϕ)Ylm∗(θ′,ϕ′).(1)\begin{aligned} G(\mathbf r,\mathbf r') &= ik \sum_{l=0}^\infty \sum_{m=-l}^l j_l(kr_<)\,h_l^{(1)}(kr_>)\, Y_l^m(\theta,\phi) Y_l^{m*}(\theta',\phi'),\\ &=\sum_{l=0}^\infty \sum_{m=-l}^l g_l(r,r')\, Y_l^m(\theta,\phi) Y_l^{m*}(\theta',\phi'). \end{aligned} \tag{1} G(r,r′)​=ikl=0∑∞​m=−l∑l​jl​(kr<​)hl(1)​(kr>​)Ylm​(θ,ϕ)Ylm∗​(θ′,ϕ′),=l=0∑∞​m=−l∑l​gl​(r,r′)Ylm​(θ,ϕ)Ylm∗​(θ′,ϕ′).​(1)

其中，

  • 球谐函数控制角度分布；
  • 球贝塞尔/汉克尔函数控制径向传播；
  • 格林函数是所有本征模态的叠加；
  • k→0k\to0k→0 时退化为静电势格林函数 1/4πr1/4\pi r1/4πr；
  • r<=min⁡(r,r′)r_< = \min(r,r')r<​=min(r,r′)，r>=max⁡(r,r′)r_> = \max(r,r')r>​=max(r,r′)；
  • gl(r,r′)=ik jl(kr<) hl(1)(kr>)g_l(r,r') = ik\,j_l(k r_<)\,h_l^{(1)}(k r_>)gl​(r,r′)=ikjl​(kr<​)hl(1)​(kr>​) 为径向格林函数；
  • Ylm(θ,ϕ)=Nl,m⋅Plm(cos⁡θ)⋅eimϕY_l^m(\theta,\phi)=N_{l,m}\cdot P_l^m(\cos\theta)\cdot e^{im\phi}Ylm​(θ,ϕ)=Nl,m​⋅Plm​(cosθ)⋅eimϕ 为球谐函数，其中 Nl,m=(−1)m2l+14π(l−m)!(l+m)!N_{l,m}=(-1)^m\sqrt{\frac{2l+1}{4\pi}\frac{(l-m)!}{(l+m)!}}Nl,m​=(−1)m4π2l+1​(l+m)!(l−m)!​​ 为归一化系数，Plm(cos⁡θ)⋅eimϕP_l^m(\cos\theta)\cdot e^{im\phi}Plm​(cosθ)⋅eimϕ 为方位量子数 mmm 的连带勒让德多项式与方位角复指数的角向函数。

上面是不考虑球对称性下的，即考虑边界情况时，完整的亥姆霍兹方程的格林函数解。

3D 情形亥姆霍兹方程的格林函数

当我们只考虑 自由空间、无边界 的 3D 情形时，可以直接求出球面波解，下面是求解过程

把问题转成相对坐标

由于系统是平移不变的，可以只考虑 r′=0\mathbf r' = 0r′=0，即：

(∇2+k2)G(r)=−δ(r)(2)(\nabla^2 + k^2) G(\mathbf r) = -\delta(\mathbf r)\tag{2} (∇2+k2)G(r)=−δ(r)(2)

设 r=∣r∣r = |\mathbf r|r=∣r∣。

利用球对称性

GGG 只依赖于 rrr，所以可以写成 G(r)G(r)G(r)。
在球坐标中：

∇2G=1r2ddr(r2dGdr)(3)\nabla^2 G = \frac{1}{r^2}\frac{d}{dr}\left(r^2 \frac{dG}{dr}\right)\tag{3} ∇2G=r21​drd​(r2drdG​)(3)

于是方程变成：

1r2ddr(r2dGdr)+k2G=−δ(r)(4)\frac{1}{r^2}\frac{d}{dr}\left(r^2 \frac{dG}{dr}\right) + k^2 G = -\delta(\mathbf r)\tag{4} r21​drd​(r2drdG​)+k2G=−δ(r)(4)

先求解右边=0的齐次方程

对 r>0r>0r>0，右边 δ=0：

d2(rG)dr2+k2(rG)=0(5)\frac{d^2 (rG)}{dr^2} + k^2 (rG) = 0\tag{5} dr2d2(rG)​+k2(rG)=0(5)

设 u=rGu = rGu=rG，解为：

u(r)=Aeikr+Be−ikr(6)u(r) = A e^{ikr} + B e^{-ikr}\tag{6} u(r)=Aeikr+Be−ikr(6)

所以：

G(r)=Aeikr+Be−ikrr(7)G(r) = \frac{A e^{ikr} + B e^{-ikr}}{r}\tag{7} G(r)=rAeikr+Be−ikr​(7)

确定常数（辐射条件）

  • eikr/re^{ikr}/reikr/r：表示向外传播的球面波；
  • e−ikr/re^{-ikr}/re−ikr/r：表示向内传播的球面波。

物理上我们要出射波（源点向外发散），即满足 Sommerfeld 辐射条件：

lim⁡r→∞r(∂G∂r−ikG)=0(8)\lim_{r\to\infty} r\left(\frac{\partial G}{\partial r}-ikG\right)=0\tag{8} r→∞lim​r(∂r∂G​−ikG)=0(8)

这要求 B=0B=0B=0，为什么？

球面波近似形式：

G(r)∼eikrr(9)G(r) \sim \frac{e^{ikr}}{r}\tag{9} G(r)∼reikr​(9)

它的径向导数是：

∂G∂r∼(ik−1r)eikrr(10)\frac{\partial G}{\partial r} \sim \left(ik - \frac{1}{r}\right)\frac{e^{ikr}}{r}\tag{10} ∂r∂G​∼(ik−r1​)reikr​(10)

把它代入表达式：

r(∂G∂r−ikG)=r[(ik−1r)eikrr−ikeikrr]=−eikrr→r→∞0.(11)r\left(\frac{\partial G}{\partial r} - ik G\right) = r \left[\left(ik - \frac{1}{r}\right)\frac{e^{ikr}}{r} - ik \frac{e^{ikr}}{r}\right] = - \frac{e^{ikr}}{r} \xrightarrow{r\to\infty} 0.\tag{11} r(∂r∂G​−ikG)=r[(ik−r1​)reikr​−ikreikr​]=−reikr​r→∞​0.(11)

所以 eikr/re^{ikr}/reikr/r 满足辐射条件。

而如果我们换成 e−ikr/re^{-ikr}/re−ikr/r：

r(∂G∂r−ikG)=r[(−ik−1r)e−ikrr−ike−ikrr]=−2ikre−ikrr=−2ike−ikr↛0.(12)r\left(\frac{\partial G}{\partial r} - ik G\right) = r \left[(-ik - \frac{1}{r})\frac{e^{-ikr}}{r} - ik\frac{e^{-ikr}}{r}\right] = -2ikr \frac{e^{-ikr}}{r} = -2ik e^{-ikr} \not\to 0.\tag{12} r(∂r∂G​−ikG)=r[(−ik−r1​)re−ikr​−ikre−ikr​]=−2ikrre−ikr​=−2ike−ikr→0.(12)

它不满足条件（因为远处波是往内收的）。

因此：

Sommerfeld 辐射条件⇒B=0.(13)\text{Sommerfeld 辐射条件} \Rightarrow B = 0.\tag{13} Sommerfeld 辐射条件⇒B=0.(13)

于是：

G(r)=Aeikrr(14)G(r) = \frac{A e^{ikr}}{r}\tag{14} G(r)=rAeikr​(14)

确定系数 AAA

在 r=0r=0r=0 附近，δ 函数起作用。将方程在一个以 r=0r=0r=0 为中心的小球体 VϵV_\epsilonVϵ​ 内积分：

∫Vϵ(∇2G+k2G) dV=−∫Vϵδ(r) dV=−1(15)\int_{V_\epsilon} (\nabla^2 G + k^2 G)\, dV = -\int_{V_\epsilon} \delta(\mathbf r)\, dV = -1\tag{15} ∫Vϵ​​(∇2G+k2G)dV=−∫Vϵ​​δ(r)dV=−1(15)

对第一项用高斯定理：

∮Sϵ∂G∂n dS+k2∫VϵG dV=−1(16)\oint_{S_\epsilon} \frac{\partial G}{\partial n}\, dS + k^2 \int_{V_\epsilon} G\, dV = -1\tag{16} ∮Sϵ​​∂n∂G​dS+k2∫Vϵ​​GdV=−1(16)

当 ϵ→0\epsilon \to 0ϵ→0，第二项趋近 0，只剩第一项。在球面上 r=ϵr=\epsilonr=ϵ，

∂G∂r=A(ikϵ−1)eikϵϵ2(17)\frac{\partial G}{\partial r} = A\frac{(ik\epsilon - 1)e^{ik\epsilon}}{\epsilon^2}\tag{17} ∂r∂G​=Aϵ2(ikϵ−1)eikϵ​(17)

积分面面积 4πϵ24\pi\epsilon^24πϵ2，于是：

∮Sϵ∂G∂n dS=4πA(ikϵ−1)eikϵ→−4πA(18)\oint_{S_\epsilon} \frac{\partial G}{\partial n}\, dS = 4\pi A(ik\epsilon - 1)e^{ik\epsilon} \to -4\pi A\tag{18} ∮Sϵ​​∂n∂G​dS=4πA(ikϵ−1)eikϵ→−4πA(18)

匹配上式：

−4πA=−1⇒A=14π(19)-4\pi A = -1 \Rightarrow A = \frac{1}{4\pi}\tag{19} −4πA=−1⇒A=4π1​(19)

得到最终结果：

G(r,r′)=eik∣r−r′∣4π∣r−r′∣(20)\boxed{G(\mathbf r, \mathbf r') = \frac{e^{ik|\mathbf r - \mathbf r'|}}{4\pi |\mathbf r - \mathbf r'|}}\tag{20} G(r,r′)=4π∣r−r′∣eik∣r−r′∣​​(20)

这就是 3D Helmholtz 方程的自由空间格林函数。

2D 情形亥姆霍兹方程的格林函数

设 r=∣r−r′∣r = |\mathbf r - \mathbf r'|r=∣r−r′∣，解的径向形式为：

1rddr(rdGdr)+k2G=−δ(r)(21)\frac{1}{r}\frac{d}{dr}\left(r\frac{dG}{dr}\right) + k^2 G = -\delta(\mathbf r)\tag{21} r1​drd​(rdrdG​)+k2G=−δ(r)(21)

齐次方程的解为 Bessel 函数：

G(r)=AJ0(kr)+BY0(kr)(22)G(r) = A J_0(kr) + B Y_0(kr)\tag{22} G(r)=AJ0​(kr)+BY0​(kr)(22)

满足“向外辐射条件”的组合是：

H0(1)(kr)=J0(kr)+iY0(kr)(23)H_0^{(1)}(kr) = J_0(kr) + iY_0(kr)\tag{23} H0(1)​(kr)=J0​(kr)+iY0​(kr)(23)

于是：

G(r,r′)=i4H0(1)(k∣r−r′∣)(24)\boxed{G(\mathbf r, \mathbf r') = \frac{i}{4} H_0^{(1)}(k|\mathbf r - \mathbf r'|)}\tag{24} G(r,r′)=4i​H0(1)​(k∣r−r′∣)​(24)

这就是 2D Helmholtz 格林函数。

1D 情形亥姆霍兹方程的格林函数

在 1D 情况下：

d2Gdx2+k2G=−δ(x−x′)(25)\frac{d^2 G}{dx^2} + k^2 G = -\delta(x-x')\tag{25} dx2d2G​+k2G=−δ(x−x′)(25)

通解为：

G(x,x′)=i2keik∣x−x′∣(26)G(x,x') = \frac{i}{2k} e^{ik|x-x'|}\tag{26} G(x,x′)=2ki​eik∣x−x′∣(26)

这一维形式可以看作是 2D、3D 格林函数的“退化版本”。

3D 情形格林函数与完整形式解的统一

格林函数的球谐展开为：

G(r,r′)=ik∑l,mjl(kr<) hl(1)(kr>) Ylm(Ω)Ylm∗(Ω′).(27)G(\mathbf r,\mathbf r') =ik\sum_{l,m} j_l(kr_<)\,h_l^{(1)}(kr_>)\, Y_l^m(\Omega)Y_l^{m*}(\Omega'). \tag{27} G(r,r′)=ikl,m∑​jl​(kr<​)hl(1)​(kr>​)Ylm​(Ω)Ylm∗​(Ω′).(27)

源在原点

现在令源点在原点，所以 r′=0r'=0r′=0，于是 r<=r′=0r_< = r'=0r<​=r′=0，因此对任意 l≥1l\ge 1l≥1，

jl(kr<)=jl(0)=0.(28)j_l(k r_<)=j_l(0)=0.\tag{28} jl​(kr<​)=jl​(0)=0.(28)

唯一存活的是：

j0(0)=1.(29)j_0(0)=1.\tag{29} j0​(0)=1.(29)

所以（27）中的和只剩：

G(r,0)=ik  j0(0)  h0(1)(kr)  Y00(Ω)  Y00∗(0).(30)G(\mathbf r,0) =ik\; j_0(0)\; h_0^{(1)}(kr)\; Y_0^0(\Omega)\; Y_0^{0*}(0).\tag{30} G(r,0)=ikj0​(0)h0(1)​(kr)Y00​(Ω)Y00∗​(0).(30)

因为 Y00=1/4πY_0^0=1/\sqrt{4\pi}Y00​=1/4π​，得到：

G(r,0)=ik h0(1)(kr)⋅14π.(31)G(\mathbf r,0) =ik\, h_0^{(1)}(kr)\cdot\frac{1}{4\pi}.\tag{31} G(r,0)=ikh0(1)​(kr)⋅4π1​.(31)

而

h0(1)(x)=−ieixx,(32)h_0^{(1)}(x)=-i\frac{e^{ix}}{x},\tag{32} h0(1)​(x)=−ixeix​,(32)

所以：

G(r,0)=eikr4πr.(33)G(\mathbf r,0)=\frac{e^{ikr}}{4\pi r}.\tag{33} G(r,0)=4πreikr​.(33)

完全等于自由空间格林函数的标准球面波形式。

源在任意

整个问题在自由空间中是各向同性的，所以一旦你知道源在原点时两种表达一致，对任意 r′\mathbf r'r′ 只不过是坐标旋转和平移，两者仍然必须一致。

用球谐加法定理：

∑m=−llYlm(Ω)Ylm∗(Ω′)=2l+14πPl(cos⁡γ),(34)\sum_{m=-l}^{l}Y_l^m(\Omega)Y_l^{m*}(\Omega') =\frac{2l+1}{4\pi}P_l(\cos\gamma),\tag{34} m=−l∑l​Ylm​(Ω)Ylm∗​(Ω′)=4π2l+1​Pl​(cosγ),(34)

带入（27）并对 mmm 求和

G(r,r′)=ik∑l=0∞2l+14πjl(kr<) hl(1)(kr>) Pl(cos⁡γ).(35)G(r,r′)= ik\sum_{l=0}^\infty \frac{2l+1}{4\pi} j_l(k r_<)\,h_l^{(1)}(k r_>)\,P_l(\cos\gamma). \tag{35} G(r,r′)=ikl=0∑∞​4π2l+1​jl​(kr<​)hl(1)​(kr>​)Pl​(cosγ).(35)

右边显然只依赖 r,r′,γr,r',\gammar,r′,γ，也就是只依赖 ∣r−r′∣|\mathbf r-\mathbf r'|∣r−r′∣，因此它必然是某个 F(∣r−r′∣)F(|\mathbf r-\mathbf r'|)F(∣r−r′∣)。

又由于在特殊情形 r′=0\mathbf r'=0r′=0 时，(2) 已经等于 eikr/(4πr)e^{ikr}/(4\pi r)eikr/(4πr)，结合方程的唯一性 ⇒ 对任意 r′\mathbf r'r′，(2) 就是

G(r,r′)=eik∣r−r′∣4π∣r−r′∣.(36)G(\mathbf r,\mathbf r')=\frac{e^{ik|\mathbf r-\mathbf r'|}}{4\pi|\mathbf r-\mathbf r'|}.\tag{36} G(r,r′)=4π∣r−r′∣eik∣r−r′∣​.(36)

总结

  • 从 PDE + 边界条件 出发构造：
    分离变量 → 球谐展开 → 径向格林函数方程 → 正则内解 jlj_ljl​、辐射外解 hl(1)h_l^{(1)}hl(1)​ → 用朗斯基调常数得 ikikik → 得到模态展开式 (1)。

  • 从物理直观球面波 出发：
    已知自由空间格林函数是 eikR/(4πR)e^{ikR}/(4\pi R)eikR/(4πR)，把它投影到球谐基底，得到同样的 gl=ikjlhl(1)g_l=ik j_l h_l^{(1)}gl​=ikjl​hl(1)​。因为两者满足同一方程和边界条件、又在一个特殊点上完全吻合，所以它们在整个空间上就是同一函数。

可以把

eikR4πR\frac{e^{ikR}}{4\pi R} 4πReikR​

看成“位置表象”的写法，而

ik∑jlhl(1)YlYl∗ik\sum j_l h_l^{(1)}Y_lY_l^* ik∑jl​hl(1)​Yl​Yl∗​

是“角动量模态表象”的写法。
